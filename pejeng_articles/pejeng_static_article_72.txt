PEJENG - Semakin hari, pertandingan di cabang voli serangkaian Porsenides IV Pejeng tampak semakin heboh saja. Lihat saja suasana di timur laut lapangan Sapta Dharma itu selalu dipadati penonton. Seperti terlihat ketika Tim Voli Kampush Banjar Puseh berhadapan dengan Tim Voli Ivong Banjar Guliang. Begitu pula saat Tim Voli Yowilas Banjar Pande versus Tim Voli Pedapdapan, para penonton tampak setia mengikuti jalannya pertandingan hingga usai.

Penampilan Tim Voli Kampush yang dimotori Gung Bhayangkara, Kale, serta sejumlah pemain The Moon Pejeng ini memang meyakinkan. Selain postur tubuhnya yang rata-rata tinggi,  mental bertandingnya juga telah tertempa di sejumlah event. Demikain pula penampilan anak-anak Ivong yang diperkuat tosser Dede Sutawan dkk, sesungguhnya juga bagus. Beberapa pemain andalannya juga punya smesh-smesh tajam. Wajarlah kalau pertandingan kedua tim ini berlangsung menarik pada malam itu.

Pada set pertama, stamina pemain kedua tim memang dalam kondisi fit. Mereka pun saling lancarkan smesh ke arah lawan. Kejar-kejaran point sempat mewarnai set pertama ini.  Kerja keras anak-anak Ivong pun kandas, setelah anak-anak Kampush lebih dahulu mengakhiri pertandingan dengan nilai 25 : 20.

Memasuki set ke dua, pertandingan ini berlangsung semakin menarik. Jual-beli smesh kembali mewarnai pertandingan yang dipimpin wasit  I, Gung Anom bersama Wasit II, Wayan Suta tersebut. Dalam pertandingan ini anak-anak Kampush dengan mudah mengakhiri pertandingan dengan poin 25 : 17.

Tertinggal dua set membuat anak-anak Guliang kehilangan semangat. Beberapa smesh yang coba dilancarkannya selalu berhasil dibendung anak-anak Kampush. Momen itu rupanya membuat anak-anak Kampush kian bergairah lalu balik melancarkan smesh-smesh tajamnya. Akhirnya Kampus kembali unggul dengan  poin 25 : 16, sekaligus menutup pertandingan dengan kemenangan 3 set berturut-turut.

Yowilas vs Pedapdapan

Sementara itu pertandingan antara Tim Voli Yowilas Banjar Pande dengan Tim Voli Pedapdapan berlangsung kurang greget. Betapa tidak, anak-anak Yowilas berhasil memenangkan pertandingan dengan mudah, 3 set berturut-turut.

Pada set awal, anak-anak Yowilas berhasil mengakhiri pertandingan dengan poin 25 : 8. Memasuki set kedua, anak-anak Pedapdapan mampu mengimbangi permainan hingga berhasil  meraih 17 poin. Namun poin itu mentok hingga Yowilas memenangkan set kedau 25 : 17.

Memang kedua tim sempat saling melancarkan smesh-smesh andalannya. Seperti yang dilakukan pemain Pedadapan Gungde Gus dkk, beberapa smeshnya sempat membuat Yowilas keteteran. Namun hal itu tidak berlangsung lama, karena setelah itu anak-anak Yowilas balik melancarkan smesh serta tipuan mematikan. Akibatnya set ketiga masih menjadi milik Yowilas yang sukses melibas Pedapdapan dengan poin 25 : 16.

Kemenangan Yowilas Banjar Pande atas Pedapdapan ini disambut meriah para supporter  setianya. Mereka berharap dalam pertandiangan selanjutnya bisa menang kembali. "Mudah-mudahan bisa sampai di final," harap Puji diiyakan supporter lainnya. (dewa suamba)
